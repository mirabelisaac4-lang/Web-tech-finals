# ACITY CONNECT

ACITY CONNECT is a smart campus marketplace and skill exchange platform for Academic City students. Students can register with an institutional email, manage a profile, post second-hand items, offer or request skills, express interest in listings, track trade requests, and allow admins to moderate platform activity.

## Feature Checklist

- Secure registration and login system
- Registration restricted to ACity email domain: `@acity.edu.gh`
- User profile page with personal information, skills offered, and skills needed
- Profile update feature
- Post item and skill listings
- Listings include title, description, category, and status
- Searchable and filterable listing feeds
- Interested button for trade requests
- Notification/request tracking on profile page
- Admin dashboard with listing moderation
- Admin can approve, mark sold/swapped, and delete inappropriate listings
- Admin can view number of listings, pending listings, and interactions

## Installation Instructions

1. Download or clone the project.
2. Open `index.html` in a browser.
3. Use the sample login details above or register with an email ending in `@acity.edu.gh`.
4. To reset demo data, clear the browser localStorage for the site and reload.
